Задание:
Дан массив(список) из 1 млн. не отсортированных постов.
Требуется найти 10 самых залайканых постов. 

Решение:
Приведено 3 варианта решения задачи. В тестах были сделаны замеры по времени работы каждого из методов.
С предсавленным заданием быстрее всех справился кастомный метод сортировки. 
В этом методе сначала отбрасываются из выборки посты с наименьшим количеством лайков, 
только после происходит сортировка методом sort() интерфейса List.
